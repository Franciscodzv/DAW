Explicación problema 6

Para el problema 6 diseñe un cajero automático con el cual puedes hacer retiros y depositos. El objeto es de tipo cajero y su constructor inicializa el saldo a $10,000. Utilizo dos métodos para poder retirar o depositar dinero y tu estado de cuenta se actualiza ua vez que retiras o depositas la cantidad que desees. Hay 3 opciones que puedes elegir en este problema, retirar = 1, depositar = 2 y salir = 0.



Preguntas a responder:

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias: Java es un lenguaje de programación y JS es un lenguaje de scripting.

Java es un lenguaje "strongly typed" y JS es mucho más relajado en cuanto a reglas de sintáxis.

Java puede correr en cualquier máquina virtual mientras que JS solo puede correr en un navegador.

Los objetos en Java son basados en clases y en JS son basados en prototipos.

Similitudes: Ambos son lenguajes orientados a objetos.

El objeto Date tiene muchos métodos entre ellos: GetFullYear(), getMonth(), getDate(), getHours(), getMinutes(), getSeconds(), getTime().

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

unShift() | Regresa la nueva longitud del arreglo.
