Cuestionario 1

¿Qué es entonces una base de datos - DB?

Una base de datos es una colección organizada de información estructurada, tipicamente guardada electronicamente en un sistema informático.

¿En qué casos es conveniente usar bases de datos?

Es conveniente usar bases de datos cuando se necesita que la información pueda ser facilmente accesada, gestionada, modificada, actualizada, controlada y organizada. Una base de datos facilita todos estos aspectos.

¿Qué es un sistema de gestión de base de datos - DBMS?

Una base de datos es normalmente controlada por un sistema de administración de base de datos o un "database management system" (DBMS). Un DBMS tiene como funciones primarias el servir como una interfaz entre el usuario y la base de datos, simultaneamente el DBMS gestiona la información, el engine de la base de datos y el esquema de la misma para así facilitar la organización y el manejo de la información dentro de la base de datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Aunque las funciones de un DBMS varían mucho, una DBMS de propósito general debe incluir:

1. Un catálogo accesible para el usuario que describa metadatos. En otras palabras, para que el usuario pueda acceder facilmente a los datos sobre la información contenida en la base de datos.

2. Abstracción e independencia de datos. Que los datos sean independientes unos de otros para facilitar el manejo de información.

3. Seguridad de datos. Que los datos no sean accesibles a todo el personal, que sea seguro actualizar datos además de que la información dentro de la base de datos no pueda ser robada.

4. Rehistro y auditoría de actividad. Saber quien está accesando a la información y que es lo que está haciendo con ella.

5. Soporte para concurrencia y transacciones. Que se puedan realizar pagos y se pueda guardar la información de pago dentro de la base de datos.

6. Soporte para autorización de acceso. Acceso a información dependiendo del rol que se tiene.

7. Soporte de acceso desde ubicaciones remotas.

8. Soporte de recuperación de datos. En caso de daño y aplicación de restricciones para garantizar que los datos sigan ciertas reglas.

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

Un proyecto donde hubiera sido conveniente utilizar una base de datos fue en el proyecto de la materia de videojuegos. Para el proyecto utilizamos muchísimos assets además de scripts, audio, etc. Creo que una base de datos donde estuviera toda esa información guardada de manera estructurada y ordenada habría sido de gran ayuda, además claro que podría accesar a cualquier datos con mucha mayor velocidad.
