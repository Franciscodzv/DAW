¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications?

AJAX es importante para el desarrollo de RIA's ya que un RIA debe correr más rápido y ser más atractivo. AJAX permite intracciones más ricas al comunicarse con los servidores sin necesidad de acciones tipo "submit".

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

Las validaciones deben hacerse del lado del servidor. EL usuario solo debe disfrutar la USX.

¿Qué es JSON?

JSON (JavaScript Object Notation) es un formato de texto para el intercambio de datos. Una ventaja de JSON sobre XML es que es más sencillo escribir un analizador sintáctico.
